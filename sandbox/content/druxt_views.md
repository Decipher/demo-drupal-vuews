# Druxt Views

https://views.druxtjs.org

> Drupal Views Vue.js components for Nuxt.js.

## Features

* `<DruxtView />` component.
* Slot and slotless theming.
* Pagination, Sorting and Filters.
* Storybook integration.

```vue
<DruxtView
  display-id="page_1"
  uuid="e530624f-cc93-42ef-8fbd-7546acc64eac"
  view-id="recipes"
/>
```
